# Thrustr Plugin System

Thrustr supports extending its functionality through [Extism](https://extism.org/) plugins. This directory contains official plugins as well as [XTP schemas](https://docs.xtp.dylibso.com/docs/concepts/xtp-schema) that can be used to generate boilerplate code for creating your own plugins.

## Prerequisites

Install the `xtp` CLI tool. See the [XTP CLI intallation guide](https://docs.xtp.dylibso.com/docs/cli/#installation)
