
# Thrustr

Thrustr is a high-performance, cross-platform and extensible video game manager and launcher that aggregates various storefronts into a single, unified interface. Thrustr is built with [GPUI](https://github.com/zed-industries/zed/tree/main/crates/gpui), a GPU accelerated UI framework powered by Rust.
## Development

Make sure the following dependencies are installed and running in your system.

### Prerequisites

 - Rust
 - [macOS only] Metal. [Instructions here.](https://github.com/zed-industries/zed/tree/main/crates/gpui#macos)

### Running

Execute `cargo run` for running the application.
